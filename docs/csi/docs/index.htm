About luga.csi

luga.csi (client side includes) is a library that let you "include" external files into an HTML document using Ajax, without requiring any kind of server-side functionality.

While server-side includes must be used whenever possible, CSI can be handy if you have a collection of HTML documents that must work even without being accessed from a webserver; downloadable documentation (like this) or CD-ROM containing XHTML files are a typical case.

Take a look at the samples to see it in action.

Syntax reference
data-luga-csi Mandatory. Point to the file you want to include. It must be a text file, the extension doesn't really matters, .inc, .txt, .xml, .htm or whatever you see fit. You can use document relative or root relative paths, or even full URLs. Due to browser's security restrictions, you can't include file from a different domain.
data-luga-after Optional. This attribute allows to specify a function that will be invoked once the include is successfully performed. See sample page

How to use it

Include the relevant JavaScript files first:

  1. <script src="jquery.min.js"></script>
  2. <script src="luga.js"></script>
  3. <script src="luga.csi.js"></script>

Then use an almost plain vanilla HTML <div> tag:

  1. <div data-luga-csi="include1.inc"></div>
