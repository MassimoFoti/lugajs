The "data-luga-after" attribute

This attribute allows to specify a function that will be invoked once the include is successfully performed.

  1. <div data-luga-csi="include1.inc" data-luga-after="afterDemo"></div>

luga.csi will pass three parameters: func(rootNode, url, response)

  1. <script>
  2. function afterDemo(rootNode, url, response) {
  3. luga.utils.displayMessage(rootNode, "CSI performed successfully. After handler executed as expected");
  4. }
  5. </script>
