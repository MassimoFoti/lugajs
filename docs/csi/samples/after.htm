The "data-lugacsi-after" attribute

This attribute allows to specify a function that will be invoked once the include is successfully performed.

<div data-lugacsi="include1.inc" data-lugacsi-after="afterDemo"></div>

luga.csi will pass three parameters: rootNode, url, response

function afterDemo(rootNode, url, response){
	luga.utils.displayMessage(rootNode, "CSI performed successfully. After handler executed as expected");
}
