The JavaScript code

Let's take a look at the code, one bit at time. First, we ensure our dataSet inherit from luga.data.HttpDataSet. We also override the dataType property. This is going to be used by jQuery.ajax(). In our case we don't want the XML's DOM, we want a plain string since we are going to use jQuery.parseXML to parse it. That's why we use "text" here: 

luga.extend(luga.data.HttpDataSet, this, [options]);
/** @type {luga.data.Rss2Dataset} */
var self = this;
/** @override */
this.dataType = "text";

The most important part is the .loadRecords() method. This method is abstract in luga.data.HttpDataSet and must be implemented by any class extending it. It's the method that gets invoked after a successfully XHR request.

We perform a few actions here. First turn the XML string into a jQuery object, using jQuery.parseXML. Than we iterate over each <item> element, turn it into an hashmap and insert inside the dataSet. Finally, we store all the metadata contained inside the <channel>, since we want to expose them as part of our context. 

/**
	* Retrieves XML data, either from an HTTP response or from a direct call
	* @param {string}   xmlStr       XML document as string. Either returned from the server or passed directly
	* @param {string}   textStatus   HTTP status. Automatically passed by jQuery for XHR calls
	* @param {object}   jqXHR        jQuery wrapper around XMLHttpRequest. Automatically passed by jQuery for XHR calls
	* @override
*/
this.loadRecords = function(xmlStr, textStatus, jqXHR){
	self.rawXml = xmlStr;
	var $xml = jQuery(jQuery.parseXML(xmlStr));
	// Add a record for each <item>
	$xml.find("item").each(function(index, element){
		self.insert(itemToHash(jQuery(this)));
	});
	setChannelMeta($xml);
};

Since we want to have a custom context, we need to override getContext() too. Usually this is not required. The code here is very similar to what's used inside luga.data.DataSet. We are just using items instead of entities to exspose our records. On top of data we add all the <channel> metadata to the context: 

/**
	* @returns {luga.data.Rss2Dataset.context}
	* @override
*/
this.getContext = function(){
	var context = {
		items: self.select(),
		recordCount: self.getRecordsCount()
	};
	var stateDesc = luga.data.utils.assembleStateDescription(self.getState());
	luga.merge(context, stateDesc);
	luga.merge(context, this.channelMeta);
	return context;
};

There are also some utility, private methods. That are specific to the RSS 2.0. We recommend you to take a look at the source code inside luga.data.Rss2Dataset
