About Luga Data

Luga Data is a data-centric library, designed to retrieve data from diverse sources, expose them using a unified API, and allow you to consume and display the data anywhere within your JavaScript and HTML code, leveraging databinding and HTML templating.

Philosophy

Developing Luga Data we keep a few clear goals in mind. We aren't looking for another MV* framework, we only want to do few things, stay focused and try to do them well.

Above anything else, we believe that simple, common tasks, should become almost trivial. At the very same time, complex tasks should be possible, even if the required functionality is not available out of the box.

History

Many ideas you will find here were inspired by Spry Data. Instead of forking that project, we decided to start afresh, since we wanted to also leverage jQuery and Handlebars. While the code was written from scratch, luga.Notifier is heavily based on the original Spry codebase and the data sorting implementation is almost the same.

Very special thanks to Kin Blas.

Unit test

Luga Data contains a comprehensive set of unit tests, developed with Jasmine.

The unit tests play a fundamental role in ensuring the code is robust and, at the very same time, provide plenty of useful information on the API.
