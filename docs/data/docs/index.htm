About Luga Data

Luga Data is a data-centric library

Getting started

In order to use Luga Data, you must include the JavaScript files. First jQuery, then Luga Core and Luga Data (in this order)

  1. <script src="js/jquery.min.js"></script>
  2. <script src="js/luga.core.min.js"></script>
  3. <script src="js/luga.data.min.js"></script>
Unit test

Luga Data contains a comprehensive set of unit tests, developed with Jasmine

The unit tests play a fundamental role in ensuring the code is robust and, at the very same time, provide plenty of useful information on the API
