Getting started

In order to use Luga Data, you must include the JavaScript files. First jQuery, then Luga Core and Luga Data (in this order)

If you want to use HTML templating, you must include Handlebars as well. 

<script src="js/jquery.min.js"></script>
<script src="js/handlebars.min.js"></script>
<script src="js/luga.core.min.js"></script>
<script src="js/luga.data.min.js"></script>

In order to become familiar with the fundamental building blocks, we recommend to start with dataSet and their API. A solid understanding of the notifier/observer is going to help you a lot, since it's a core concept in Luga Data.

If instead you are the kind of person who prefers to get up and running immediately, you may dive directly into, regions, databinding to dynamically generate your GUI. Or quickly assemble a master-details page.
