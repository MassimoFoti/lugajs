Region and Handlebars templates

By default Luga regions use Handlebars for templating. The template itself can be inline, embed, or external.

In case you have special needs or would like to use a different templating engine, you can write your own region implementation.

Inline template

Inline template is the simplest option, just put Handlebars's code inside the region's HTML tag:

Warning! You may encounter problems if your inline template contains <table> tags. Certain browsers try to parse and render it. If you need tables, you better use embed or external templates. 
<div data-lugads-region="inlineTempalte" data-lugads-datasource="ladiesDs">
	<ul class="luga-docs-list">
		{{#each context}}
		<li>{{firstName}} {{lastName}}</li>
		{{/each}}
	</ul>
</div>
    {{#each context}}
  • {{firstName}} {{lastName}}
    • {{/each}}
Embed template

An embed template store Handlebar's code in a <script> tag inside the current page. This way it will not be visible and parsed by the browser before the JavaScript code gets executed, it will not be indexed by search engines either.

It's critical to add the appropriated type attribute to the <script> tag, this way the browser will not execute it as JavaScript code: 

<script id="ladiesTemplate" type="text/x-handlebars-template">
	<ul class="luga-docs-list">
		{{#each context}}
		<li>{{firstName}} {{lastName}}</li>
		{{/each}}
	</ul>
</script>

Then you use the data-lugads-template custom attribute inside the region: 

<div data-lugads-region="embedTemplate" data-lugads-datasource="ladiesDs" data-lugads-template="ladiesTemplate"></div>
External template

For better reusability you can store templates inside external files and reference them in a <script> tag: 

<script id="ladiesTable" type="text/x-handlebars-template" src="../templates/ladies-table.hbs"></script>

Then you use the data-lugads-template custom attribute inside the region, just like you do for embedded templates: 

<div data-lugads-region="externalTemplate" data-lugads-datasource="ladiesDs" data-lugads-template="ladiesTable"></div>
