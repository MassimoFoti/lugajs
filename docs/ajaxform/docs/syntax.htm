Syntax reference

The complete syntax reference for HTML5 custom data attributes is included below

Form tag
data-lugajax-form Required. Set it to true in order to submit the form using Ajax
data-lugajax-action

Optional. URL to where the form will be send

If not specified, the value inside the form's action attribute will be used
data-lugajax-method

Optional. HTTP's method to be used

If not specified, the value inside the form's method attribute will be used
data-lugajax-timeout

Optional. Timeout to be used during the HTTP call (milliseconds). Default to 30000
data-lugajax-success

Optional. Name of the function to be invoked if the form is successfully submitted. Default to luga.ajaxform.handlers.replaceForm
data-lugajax-error

Optional. Name of the function to be invoked if the HTTP call failed. Default to luga.ajaxform.handlers.errorAlert
data-lugajax-successmsg

Optional. Message that will be displayed to the user if the form is successfully submitted. Default to "Thanks for submitting the form"
data-lugajax-errormsg

Optional. Message that will be displayed to the user if the HTTP call failed. Default to "Failed to submit the form"
data-lugajax-before

Optional. Name of the function to be invoked before the form is send. Default to null
data-lugajax-after

Optional. Name of the function to be invoked after the form is send. Default to null
