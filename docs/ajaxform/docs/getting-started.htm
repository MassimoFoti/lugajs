Getting started

In order to turn a form into an Ajaxform, you must include the JavaScript files. First jQuery, then Luga Core and Luga Ajaxform (in this order) 

<script src="js/jquery.min.js"></script>
<script src="js/luga.core.min.js"></script>
<script src="js/luga.ajaxform.min.js"></script>

Then add the data-lugajax-form="true" attribute to the form tag

<form action="../action.json" data-lugajax-form="true">
	<input type="text" name="name"/>
	<input type="submit" value="Submit"/>
</form>
Basic form
Warning!
  • Please note Ajaxform can't handle file upload!
  • Don't use "action" or "method" as field name! Certain browsers will mess up on this
  • Keep in mind, XMLHttpRequest expects UTF-8 and will force the response to UTF-8. Any non-UTF-8 charset you send downstream may be turned into rubbish
