Basic form

Getting started

In order to turn a form into an ajaxform, we must include the JavaScript files. First jQuery, then Luga Core and Luga Ajaxform (in this order)

Then add the data-lugajax-form="true" attribute to the form tag.

  1. <script src="js/jquery.min.js"></script>
  2. <script src="js/luga.core.min.js"></script>
  3. <script src="js/luga.ajaxform.min.js"></script>
  1. <form action="../action.json" data-lugajax-form="true"">
  2. <input type="text" name="name" />
  3. < type="submit" value="Submit"/>
  4. </form>
