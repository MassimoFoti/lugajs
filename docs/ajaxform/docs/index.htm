About Luga Ajaxform

luga.ajaxform is a JavaScript library that can turn any HTML <form> into a form that get submitted by XHR, without the need to reload the page

In the background, whenever a form is submitted, Ajaxform intercepts the submission, serializes the values of the fields, and then issues an HTTP request to the relevant action page using XHR

HTTP requests issued by Ajaxform will use "luga.ajaxform" as the value of the "X-Requested-With" field in the HTTP header. This will allow server-side scripts to be aware if the form submission is coming from XHR or not

Ajaxform allows progressive enhancement, users without JavaScript can still use the form in a traditional way; users with JavaScript will get a more sophisticated experience

Unit test

Luga Ajaxform contains a comprehensive set of unit tests, developed with Jasmine

The unit tests play a fundamental role in ensuring the code is robust and, at the very same time, provide plenty of useful information about the API

History

Luga Ajaxform evolved out of TMT Ajaxform, an XHTML-centric library originally written in 2005. While the design principles are pretty much the same and the syntax is similar, the implementation is quite different.

Unlike the older library, Luga Ajaxform takes advantage of jQuery and it's distributed under an open source license
