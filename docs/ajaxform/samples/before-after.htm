Before/after handlers

If you want to execute some code before or after the form is send, you can use the data-lugajax-before and data-lugajax-after custom attributes 

<form action="../action.json" data-lugajax-form="true" data-lugajax-after="handlers.alertAfter"
	data-lugajax-before="handlers.alertBefore">
	<input type="text" name="name"/>
	<input type="submit" value="Submit"/>
</form>

Matching existing JavaScript code: 

handlers = {};
handlers.alertBefore = function(formNode){
	alert("I am executed before the form is send");
};
handlers.alertAfter = function(formNode){
	alert("I am executed after the form is send");
};
