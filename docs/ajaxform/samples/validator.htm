Integrating with Luga Validator

If you want to first use Luga Validator and then send the form using Ajaxform, you can leverage the data-lugavalidator-after attribute and the build-in luga.ajaxform.handlers.afterValidation function

  1. <form data-lugavalidator-validate="true" data-lugavalidator-after="luga.ajaxform.handlers.afterValidation">
  2. <input type="text" name="name" />
  3. < type="submit" value="Submit"/>
  4. </form>
