Together with Luga Validator

If you want to first use Luga Validator and then send the form using Ajaxform, you can leverage the data-lugavalidator-after attribute and the build-in luga.ajaxform.handlers.afterValidation function 

<form action="../action.json"
	data-lugavalidator-validate="true"
	data-lugavalidator-after="luga.ajaxform.handlers.afterValidation">
	<input type="text" name="name"/>
	<input type="submit" value="Submit"/>
</form>
First validate, then send by XHR
