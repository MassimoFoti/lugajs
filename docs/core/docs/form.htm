luga.form.toHash(rootNode, demoronize)
Returns a JavaScript object containing name/value pairs from fields contained inside a given root node. Only fields considered successful are returned:
http://www.w3.org/TR/REC-html40/interact/forms.html#h-17.13.2
Values of multiple checked checkboxes and multiple select are included as a single entry, with array value
Parameters
rootNode jQuery jQuery object wrapping the root node
demoronize boolean If set to true, MS Word's special chars are replaced with plausible substitutes. Default to false
Returns
A JavaScript object containing name/value pairs 
<form id="basicValue">
	<input type="text" name="firstname" value="ciccio"/>
	<input type="text" name="lastname" value="pasticcio"/>
	<input type="radio" name="radio" value="yes" checked="checked"/>
	<input type="radio" name="radio" value="no"/>
	<input type="submit"/>
</form>
luga.form.toHash(jQuery("#basicValue")
// => { person: { firstname: "ciccio", lastname: "pasticcio", }, radio: "yes" }
luga.form.toQueryString(rootNode, demoronize)
Returns a URI encoded string of name/value pairs from fields contained inside a given root node. Only fields considered successful are returned:
http://www.w3.org/TR/REC-html40/interact/forms.html#h-17.13.2
Parameters
rootNode jQuery jQuery object wrapping the root node
demoronize boolean If set to true, MS Word's special chars are replaced with plausible substitutes. Default to false
Returns
A URI encoded string
