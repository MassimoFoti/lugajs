Radio buttons

Radio buttons are a special kind of field, since they come in groups, sharing the same name but having different values.

Luga Validator validates a set of radios with the same name as a unique entity. So, inside a set of radios, you can define the validation attributes only once (if you specify attributes with conflicting values, the last one wins): 

<input name="drink" type="radio" class="boxes" value="Nicole" data-lugavalidator-required="true"/>
<input name="drink" type="radio" class="boxes" value="Kate"/>
