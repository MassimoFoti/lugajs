Radio buttons

Radio buttons are a special kind of field, since they come in groups, sharing the same name but having different values.

Luga Validator validates a set of checkboxes with the same name as a unique entity. So, inside a set of checkboxes, you can define the validation attributes only once (if you specify attributes with conflicting values, the last one wins):

  1. <input name="drink" type="radio" class="boxes" value="Nicole" data-luga-required="true" />
  2. <input name="drink" type="radio" class="boxes" value="Kate" />
