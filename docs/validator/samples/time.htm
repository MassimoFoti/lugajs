data-lugavalidator-pattern

This sample shows the data-lugavalidator-pattern attribute in action.

Valid time in HH:MM format.

Meeting form

Start time (required)
End time (required)
Lunch time (optional)
  1. <form data-lugavalidator-validate="true">
  2. <input type="text" name="startTime" data-lugavalidator-required="true" data-lugavalidator-pattern="time" data-lugavalidator-errorclass="invalid" data-lugavalidator-message="Please insert start time in HH:MM format" />
  3. <input type="text" name="endTime" data-lugavalidator-required="true" data-lugavalidator-pattern="time" data-lugavalidator-errorclass="invalid" data-lugavalidator-message="Please insert end time in HH:MM format" />
  4. <input type="text" name="lunchTime" data-lugavalidator-pattern="time" data-lugavalidator-errorclass="invalid" data-lugavalidator-message="Please insert lunch time in HH:MM format" />
  5. </form>
