Use luga.validator.handlers.errorBox as error handler

By default the validator uses JavaScript alert boxes to display errors to the user. If you want to display errors in a more user friendly way, use "luga.validator.handlers.errorBox" as error handler

<form data-lugavalidator-validate="true" data-lugavalidator-error="luga.validator.handlers.errorBox">

If you need more flexibility, you can create a custom handler.

Styling and formatting for error box are specified inside an external CSS file. You can tweak and modify it to suit your needs.

The value for the default error handler is stored inside luga.validator.CONST.HANDLERS.FORM_ERROR. You can override it with something that fits your needs

luga.validator.CONST.HANDLERS.FORM_ERROR = "luga.validator.handlers.errorBox";
