<select> tag

There are two different attributes that apply only to drop-down lists created created using the <select> tag.

data-luga-invalidindex prevents selection of an entry on a given position (zero based):

  1. <select name="lady" data-luga-invalidindex="0">

data-luga-invalidvalue prevents selection of an entry with a given value.:

  1. <select name="dish" data-luga-invalidvalue="Crepes">
