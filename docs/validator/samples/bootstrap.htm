Use Bootstrap's validation states

Use Bootstrap validation states

By default the validator uses JavaScript alert boxes to display errors to the user. If we want to use Bootstrap's validation states instead, use "luga.validator.handlers.bootstrap" as error handler. See Bootstrap's docs for additional info on validation states

  1. <form data-lugavalidator-validate="true" data-lugavalidator-error="luga.validator.handlers.bootstrap">

Override default error handler

The value for the default error handler is stored inside luga.validator.CONST.HANDLERS.FORM_ERROR. You can override it with the Bootstrap handler

  1. <script>
  2. luga.validator.CONST.HANDLERS.FORM_ERROR = "luga.validator.handlers.bootstrap";
  3. </script>
