data-lugavalidator-equalto

This sample shows the data-lugavalidator-equalto attribute in action.

To prevent the user from including spaces in their username, we use data-lugavalidator-pattern="alphanumeric"

The value of the data-lugavalidator-equalto attribute must be the id of an existing field in the current document.

Registration form

Username (required, alphanumeric)
Password (required)
Repeat Password (required)
  1. <form data-lugavalidator-validate="true">
  2. <input type="text" name="username" data-lugavalidator-required="true" data-lugavalidator-pattern="alphanumeric" data-lugavalidator-errorclass="invalid" data-lugavalidator-message="Please insert a username using only numbers and letters" />
  3. <input type="password" name="password1" id="password1" data-lugavalidator-required="true" data-lugavalidator-errorclass="invalid" />
  4. <input type="password" name="password2" id="password2" data-lugavalidator-required="true" data-lugavalidator-errorclass="invalid" data-lugavalidator-equalto="password1" data-lugavalidator-message="Please insert two matching passwords" />
  5. </form>
