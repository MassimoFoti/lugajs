data-luga-equalto

This sample shows the data-luga-equalto attribute in action.

To prevent the user from including spaces in their username, we use data-luga-pattern="alphanumeric"

The value of the data-luga-equalto attribute must be the id of an existing field in the current document.

Registration form

Username (required, alphanumeric)
Password (required)
Repeat Password (required)
  1. <form data-luga-validate="true">
  2. <input type="text" name="username" data-luga-required="true" data-luga-pattern="alphanumeric"
  3. data-luga-errorclass="invalid" data-luga-message="Please insert a username using only numbers and letters" />
  4. <input type="password" name="password1" id="password1" data-luga-required="true" data-luga-errorclass="invalid" />
  5. <input type="password" name="password2" id="password2" data-luga-required="true" data-luga-errorclass="invalid"
  6. data-luga-equalto="password1" data-luga-message="Please insert two matching passwords" />
  7. </form>
