Getting started

In order to enable validation, you must include the JavaScript files. First jQuery, then Luga Common and Luga Validator (in this order) 

<script src="js/jquery.min.js"></script>
<script src="js/luga.common.min.js"></script>
<script src="js/luga.validator.min.js"></script>

Then add the data-lugavalidator-validate="true" attribute to the form tag

<form data-lugavalidator-validate="true">
	<input type="text" name="name"
	data-lugavalidator-required="true"
	data-lugavalidator-errorclass="invalid"
	data-lugavalidator-message="Invalid!"/>
	<input type="submit" value="Submit"/>
</form>
