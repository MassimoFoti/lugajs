Basic form

Getting started

In order to enable the validator we must include the JavaScript files. First jQuery, then Luga Core and Luga Validator (in this order)

Then add the data-luga-validate attribute to the form tag. Without it the validator parser will completely skip our form.

From now on we can take advantage of all the other attributes that can be used for input fields.

  1. <script src="js/jquery.min.js"></script>
  2. <script src="js/luga.core.js"></script>
  3. <script src="js/luga.validator.js"></script>
  1. <form data-luga-validate="true">
  2. <input type="text" name="name" data-luga-required="true" data-luga-errorclass="invalid" data-luga-message="Invalid!"/>
  3. <input type="submit" value="Submit"/>
  4. </form>
