Better UI for validation
Use Bootstrap's validation states

By default the validator uses JavaScript alert boxes to display errors to the user. If you want to use Bootstrap's validation states instead, use "luga.validator.handlers.bootstrap" as error handler. 

<form data-lugavalidator-validate="true" data-lugavalidator-error="luga.validator.handlers.bootstrap">

The value for the default error handler is stored inside luga.validator.CONST.HANDLERS.FORM_ERROR. You can override it the Bootstrap handler:

luga.validator.CONST.HANDLERS.FORM_ERROR = "luga.validator.handlers.bootstrap";
