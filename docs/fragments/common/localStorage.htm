luga.localStorage.retrieve(root, path)
Retrieve from localStorage the string corresponding to the given combination of root and path. Returns undefined if nothing matches the given root/path
Arguments
root {String} Top-level key inside localStorage
path {String} Dot-delimited string
Return type
{*|undefined}
Examples 
luga.localStorage.persist("myRoot", "test.name", "Ciccio");
luga.localStorage.retrieve("myRoot", "test.name"); // => "Ciccio"
luga.localStorage.persist(root, path, value)
Persist the given string inside localStorage, under the given combination of root and path. The ability to pass a dot-delimited path allow to protect the information from name clashes
Arguments
root {String} Top-level key inside localStorage
path {String} Dot-delimited string
value {String} String to be persisted
Return type
{String}
Examples 
luga.localStorage.persist("myRoot", "test.name", "Ciccio");
