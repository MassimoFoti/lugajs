Date validation

There are 14 built-in date formats.

If none of the built-in formats satisfy your needs, you can add your own pretty easily.

Min/Max Date

Please note that whenever you pick a date pattern different than the default one (YYYY-MM-DD) the same pattern is enforced on both the field value and the value used for mindate/maxdate attributes 

<form data-lugavalidator-validate="true">
	<div class="form-group">
		<label for="date15">Date 15: Date must be 21th century or above (more than 2000-01-01, default pattern)</label>
		<input type="text" class="form-control" name="date15" id="date15"
		data-lugavalidator-errorclass="invalid"
		data-lugavalidator-message="Date 15: please insert a valid 21th century or above date using
		the YYYY-MM-DD format"
		data-lugavalidator-mindate="2000-01-01"
		value="2008-12-31"/>
	</div>
	<div class="form-group">
		<label for="date16">Date 16: Date must be in 20th century (less than 1999-12-31, more than 1900-01-01, default
		pattern)</label>
		<input type="text" class="form-control" name="date16" id="date16"
		data-lugavalidator-errorclass="invalid"
		data-lugavalidator-message="Date 16: please insert a valid 20th century date using the YYYY-MM-DD format"
		data-lugavalidator-mindate="1900-01-01"
		data-lugavalidator-maxdate="1999-12-31"
		value="1969-08-09"/>
	</div>
	<div class="form-group">
		<label for="date17">Date 17: Date must be 21th century or above
		(more than 2000-01-01, MM/DD/YYYY pattern)</label>
		<input type="text" class="form-control" name="date17" id="date17"
		data-lugavalidator-errorclass="invalid"
		data-lugavalidator-required="true"
		data-lugavalidator-message="Date 17: please insert a valid 21th century or above date using the MM/DD/YYYY format"
		data-lugavalidator-mindate="01/01/2000"
		data-lugavalidator-datepattern="MM/DD/YYYY"/>
	</div>
	<div class="form-group">
		<input type="submit" value="Submit"/>
	</div>
</form>
Change default date pattern

Default date pattern is "YYYY-MM-DD". You can set a different default using JavaScript: 

luga.validator.CONST.DEFAULT_DATE_PATTERN = "M/D/YYYY";
