luga.data.DataSet

The dataSet is the most important object in Luga Data, you need to understand it well in order to use the library to its full potential. At its core the dataSet holds an array of objects and let you access them using a common API. It also support binding and HTML templating.

In order to instantiate an empty dataSet you just call its constructor. The only required parameter is the uuid, an identifier that must be unique and can be used for binding data to regions. 

var testDs = new luga.data.DataSet({uuid: "test"});

Now you can add one record to it: 

testDs.insert({
	"firstName": "Nicole",
	"lastName": "Kidman"
});

You can keep adding data, passing an array of objects, to load them all in one single swap: 

testDs.insert([
	{ "firstName": "Kate", "lastName": "Beckinsale" },
	{ "firstName": "Jennifer", "lastName": "Connelly" },
	{ "firstName": "Salma", "lastName": "Hayek" }
]);

Instead of adding data at run-time, you can add all of them upfront, passing them to the constructor: 

var ladiesData = [
	{ "firstName": "Nicole", "lastName": "Kidman" },
	{ "firstName": "Kate", "lastName": "Beckinsale" },
	{ "firstName": "Jennifer", "lastName": "Connelly" },
	{ "firstName": "Salma", "lastName": "Hayek" }
];
var ladiesDs = new luga.data.DataSet({
	uuid: "ladies",
	records: ladiesData
});

While a dataSet may reminds you of similar objects you used in Java, C# or PHP for managing data from RDBMS, it's quite a different beast, since it's not limited to tabular data. You can use it to handle a collection of database-like name/value pairs, but you can also use any JSON object, containing more structured, tree-like data: 

var jazzMenData = [
	 {
		 "person": {
			 "firstName": "John",
			 "lastName": "Coltrane"
		 },
		 "country": "USA"
	 },
	 {
		 "person": {
			 "firstName": "Sonny",
			 "lastName": "Rollins"
		 },
		 "country": "USA"
	 },
	 {
		 "person": {
			 "firstName": "Tete",
			 "lastName": "Montoliu"
		 },
		 "country": "Spain"
	 },
	 {
		 "person": {
			 "firstName": "Dexter",
			 "lastName": "Gordon"
		 },
		 "country": "USA"
	 }
];
var jazzDsDs = new luga.data.DataSet({
	 uuid: "jazzMen",
	 records: jazzMenData
});

Now that you have a dataSet, loaded with data, it's time to move to the next step and try binding.

Heads up!
luga.data.DataSet is the base dataSet class, other classes inherit from it. Very often you may want to consume JSON data from REST endpoint, loading data from an URL. In such cases you should use a JSON DataSet. If your data is in XML format instead, use a XML DataSet.
