Using traits inside regions

Traits are special features available inside regions. They are attached to the DOM after the region render its HTML. Traits add capabilities to your templates and can be expressed as custom HTML attributes. There are build-in traits, but you can write your custom traits too.

Attribute Description
data-lugaregion-setrowid Automatically set the clicked row as current row. Essential for master-details. See example.
data-lugaregion-setrowindex Same as above, just use row's index instead of rowId.
data-lugaregion-select Apply the given CSS class to the HTML tag that matches the currently selected row. Useful for tracking selection. See example.
data-lugaregion-sort Sort a dataSet using the given colun name. See example.
