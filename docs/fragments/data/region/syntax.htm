Syntax reference

The complete syntax reference for HTML5 custom data attributes is included below

Attribute Description
data-lugaregion Required. Set it to true in order to turn an HTML tag into a region.
data-lugaregion-datasource-uuid Required. Must match the uuid of an existing dataSource (either dataSet or detailsSet). Enables databinding from the dataSource to the region (read more).
data-lugaregion-template-id Optional. Must match the id attribute of an existing <script> tag containing template code (read more).
data-lugaregion-traits Optional. Comma-delimited list of function names to be used as custom trait/s (read more).
data-lugaregion-type Optional. Name of an existing JavaScript constructor function for custom region (read more). For default rendering, using Handlebars as template engine, there is no need to specify it.
